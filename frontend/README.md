# CodeIn 정적 프로토타입 (멀티페이지)

## 포함 페이지
- index.html (홈)
- notice.html (공지)
- boards.html (게시판 - 스텁)
- calendar.html (캘린더)
- gallery.html (갤러리)
- mypage.html (마이페이지/로그인 스텁)
- contest.html (코딩테스트 슬롯 - 준비중)

## 실행(권장)
```bash
cd codec_in_static_multipage
python -m http.server 8000
```
